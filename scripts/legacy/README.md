# Legacy Scripts

This directory contains auxiliary / historical runners that are kept for
provenance, visualization, or one-off exploratory workflows.

They are not part of the canonical paper-reproduction path described in
`REPRODUCE.md`.

Typical examples here are:

- Google Colab runners
- atlas / OBJ export utilities
- phase-history visualization scripts
- superseded high-precision or PSLQ experiments

If a script in this directory is still needed for public-facing
reproducibility, it should be promoted back into `scripts/` only after its
paths, imports, and documentation have been rewritten to the final repo layout.
